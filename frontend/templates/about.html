    Research Overview

    About the Project

    A hybrid deep learning framework for automated brain stroke detection from medical images.

    Project Summary

    This project successfully designed, implemented, and evaluated a hybrid deep learning framework for the automated detection of brain stroke from medical images. Motivated by the critical need for rapid and accurate diagnostic aids, this research focused on comparing two distinct hybrid models to identify an optimal balance between performance and efficiency.

    M1

    DenseNet121 + SVM

    91%
    Detection Accuracy

    Combines a powerful DenseNet121 deep feature extractor with a robust Support Vector Machine classifier. The CNN extracts rich feature representations which the SVM classifies with high precision.

    M2

    VGG16 + MLP + SMOTE

    93%
    Detection Accuracy

    Addresses class imbalance using SMOTE at the feature level, then trains a Multi-Layer Perceptron on the balanced VGG16 feature set. Achieves superior accuracy and better minority-class recall.

    Key Findings

    The comparative analysis demonstrates that both hybrid models are highly effective for stroke detection. However, the VGG16-MLP model augmented with SMOTE showed a meaningful performance advantage, highlighting the importance of addressing class imbalance in medical imaging systems. The current deployment uses MobileNetV2 with Grad-CAM for real-time explainable inference.
